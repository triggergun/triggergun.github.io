# Androidеёёз”Ёзҡ„еҹәзЎҖеёғеұҖе®№еҷЁ

Android зҡ„UIеҸҜд»ҘеҲҶдёәдёӨзұ»пјҢдёҖзұ»еҸ«еҒҡViewGroupе®№еҷЁпјҢдёҖзұ»еҸ«еҒҡViewи§ҶеӣҫгҖӮ

  • Viewи§ҶеӣҫпјҡпјҲTextViewгҖҒButtonгҖҒlmageViewпјүйғҪжҳҜеёёз”Ёеёёи§Ғзҡ„и§ҶеӣҫгҖӮ

  • ViewGroupе®№еҷЁпјҡеҶ…йғЁеҸҜд»ҘжүҝиҪҪгҖҒж”ҫзҪ®гҖҒж·»еҠ Viewи§ҶеӣҫгҖӮ

# еҹәзЎҖзҡ„еёғеұҖе®№еҷЁ

LinearLayoutзәҝжҖ§еёғеұҖпјҡжЁӘзқҖжҲ–з«–зқҖжҢүйЎәеәҸжҺ’еҲ—гҖӮ

RelativeLayoutзӣёеҜ№еёғеұҖпјҡиө·е§Ӣеқҗж Үж—¶еұҸе№•е·ҰдёҠи§’пјҢд»ҘеҗҢзә§жҲ–дёҠзә§дёәеҸӮиҖғзі»е®ҡдҪҚдҪҚзҪ®гҖӮ

FrameLayoutеё§еёғеұҖпјҡеғҸеҚғеұӮйҘјдёҖж ·пјҢдёҖеұӮеҺӢзқҖдёҖеұӮгҖӮ

ConstraintLayoutзәҰжқҹеёғеұҖпјҡgoogleдәҺ2016е№ҙж–°еҸ‘еёғзҡ„дёҖз§ҚеёғеұҖж–№ејҸпјҢе®ғдёҚеңЁandroidзҡ„еҹәзЎҖapiеҢ…йҮҢйңҖиҰҒйўқеӨ–еј•е…ҘгҖӮ

Aoselutetayoutз»қеҜ№еёғеұҖ(д»ҘеұҸе№•е·ҰдёҠи§’дёәеҸӮиҖғзі»пјҢе®ҡдҪҚиҮӘе·ұзҡ„дҪҚзҪ®пјҢд»Һandroid 2.2зүҲжң¬еҗҺеәҹејғ)

GridLayoutзҪ‘ж јеёғеұҖ(еҸҜд»ҘжҢҮе®ҡиЎҢж•°еҲ—ж•°пјҢеӯҗжҺ§д»¶иҮӘеҠЁж №жҚ®иЎҢеҲ—ж•°иҝӣиЎҢеҲҶй…ҚдҪҚзҪ®пјҢдәҺandroid 4.0еҗҺж–°еўһиҝӣapiдёӯ)

# RelativeLayout

зӣёеҜ№еёғеұҖеңЁж‘Ҷж”ҫеӯҗи§ҶеӣҫдҪҚзҪ®ж—¶пјҢжҢүз…§жҢҮе®ҡзҡ„еҸӮиҖғзі»жқҘж‘Ҷж”ҫеӯҗи§Ҷеӣҫзҡ„дҪҚзҪ®пјҢй»ҳи®Өд»ҘеұҸе№•е·ҰдёҠи§’(0,0)дҪҚзҪ®дҪңдёәеҸӮиҖғзі»ж‘Ҷж”ҫдҪҚзҪ®гҖӮ

зӣёеҜ№дәҺзҲ¶е…ғзҙ  7дёӘеёёз”ЁеұһжҖ§

еұһжҖ§ еҸҜйҖүеҖј иҜҙжҳҺ
layout _alignParentTop true/false жҳҜеҗҰи®©жҺ§д»¶зӣёеҜ№дәҺзҲ¶е®№еҷЁйЎ¶йғЁеҜ№йҪҗ
layout_alignParentBottom true/false жҳҜеҗҰи®©жҺ§д»¶зӣёеҜ№дәҺзҲ¶е®№еҷЁеә•йғЁеҜ№йҪҗ
layout_alignParentLeft true/false жҳҜеҗҰи®©жҺ§д»¶зӣёеҜ№дәҺзҲ¶е®№еҷЁе·Ұиҫ№еҜ№йҪҗ
layout_alignParentRight true/false жҳҜеҗҰи®©жҺ§д»¶зӣёеҜ№дәҺзҲ¶е®№еҷЁеҸіиҫ№еҜ№йҪҗ
layout_centerHorizontal true/false зӣёеҜ№зҲ¶е®№еҷЁж°ҙе№іеұ…icalзӨә
layout_centerVertical true/false зӣёеҜ№зҲ¶е®№еҷЁеһӮзӣҙеұ…дёӯжҳҫзӨә
centerInParent true/false зӣёеҜ№зҲ¶е®№еҷЁеұ…дёӯжҳҫзӨә

зӣёеҜ№дәҺе…„ејҹе…ғзҙ  4дёӘеёёз”ЁеұһжҖ§

еұһжҖ§ еҸҜйҖүеҖј иҜҙжҳҺ
layout above @+id/ жҢҮе®ҡеңЁйӮЈдёӘжҺ§д»¶зҡ„дёҠдҫ§
layout below @+id/ жҢҮе®ҡеңЁйӮЈдёӘжҺ§д»¶зҡ„дёҠдҫ§
android:layout toLeftOf @+id/ жҢҮе®ҡеңЁйӮЈдёӘжҺ§д»¶зҡ„е·Ұдҫ§
android:layout toRightOf @+id/ жҢҮе®ҡеңЁйӮЈдёӘжҺ§д»¶зҡ„еҸідҫ§

зӣёеҜ№дәҺе…„ејҹе…ғзҙ зҡ„еҜ№йҪҗж–№ејҸ

layout _alignLeft @+id/ иҜҘжҺ§д»¶зҡ„е·Ұиҫ№жІҝдёҺжҢҮе®ҡжҺ§д»¶зҡ„е·Ұиҫ№еҜ№йҪҗ

layout alignTop @+id/ иҜҘжҺ§д»¶зҡ„дёҠиҫ№жІҝдёҺжҢҮе®ҡжҺ§д»¶зҡ„дёҠиҫ№жІҝеҜ№йҪҗ
layout alignBottom @+id/ иҜҘжҺ§д»¶зҡ„дёӢиҫ№жІҝдёҺжҢҮе®ҡжҺ§д»¶зҡ„дёӢиҫ№жІҝеҜ№йҪҗ
