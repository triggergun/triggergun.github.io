# 解决问题

如何删除多余的IP地址呢？如不需要192.168.5.13这个地址。这里也就表示一台主机可以有多个IP地址。

方法1）通过命令行删除

sudo ip addr del 192.168.5.13/24 dev ens33
1

还有其他方式可以去问AI。这里只是抛砖引玉。

# 解决后的结果

可以看到ens33网卡只有一个ip地址绑定了。
