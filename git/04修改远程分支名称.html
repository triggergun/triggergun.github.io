# 01）查看本地分支

git branch
1

# 02）修改本地分支

 git branch -m 旧分支名 新分支名
1

再次查看已经修改了。

# 03）删除远程分支

git push --delete origin 旧分支名
1

# 04）将新分支名推上去

git push origin 新分支名
1

# 05）将新本地分支和远程相连 ：

 git branch --set-upstream-to origin/新分支名
