# vuepress常用语法

# 01）提示块

这个叫：自定义容器。

语法：

:::tip

:::
1
2
3

怎么写：

:::tip
resultMap：结果映射·标签

- type属性：数据表数据映射具体的java实体类的全路径名。

result：具体映射属性标签，属性如下：

- column：要与表中的字段名对应。
 - property：要与java的实体对象类的成员属性变量名对应
:::

# 03)字标粗&字标颜色

  • 字标粗：使用b标签，不要用二个*表示。

    <b>xxx</b>
**xxx**   [vuepress不支持]

  • 字标颜色：标记颜色用font标签。

    <font color="red">xxxxxxx</font>
