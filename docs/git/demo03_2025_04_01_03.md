# git命令查看本地分支

| 命令             | 效果                        |
| :--------------- | :-------------------------- |
| `git branch`     | 仅查看本地分支              |
| `git branch -v`  | 本地分支 + 最后一次提交信息 |
| `git branch -vv` | 本地分支 + 远程跟踪分支关系 |
| `git branch -a`  | 本地 + 远程所有分支         |
| `git branch -r`  | 仅查看远程分支              |

### **注意事项**

如果分支未显示，可能是本地尚未拉取远程分支，需先运行：

```shell
# 同步远程分支信息
git fetch origin 
```
