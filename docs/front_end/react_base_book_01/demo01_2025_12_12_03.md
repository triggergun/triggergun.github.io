# 第一个react项目



## 创建项目

```shell
npm create vite@latest hacker-stories -- --template react
```

![image-20260125121151781](demo01_2025_12_12_03.assets/image-20260125121151781.png)



## 启动react项目

```shell
cd hacker-stories
```

![image-20260125184207678](demo01_2025_12_12_03.assets/image-20260125184207678.png)

安装项目依赖：

```shell
npm install
```

终端输出的结果：

```shell
PS E:\react_code_01\hacker-stories> npm install

up to date in 823ms

33 packages are looking for funding
  run `npm fund` for details
```



运行开发环境：

```shell
npm run dev
```
