# rust命令总结

- 下面的命令一定熟悉。【不要求记住】



## 构建命令

| 命令             | 描述       |
| ---------------- | ---------- |
| cargo new 项目名 | 创建项目   |
| cargo build      | 构建项目   |
| cargo run        | 运行并编译 |
| cargo check      | 检查代码   |
