# kafka必须明确的概念知识点



## 消费者组概念

消费者组（consumer group）可以有多个消费者。比如：A组里面有A1、A2一共2个消费者。（一对多）

在同一个消费者组里面的消费者有一个约束条件。

1. 一个分区（partition）只能被一个消费者组里面的某一个消费者进行消费。



## 为什么kafka要依赖zookeeper呢？

答：这个和kafka的版本有关系。

0.9之前是依赖zookeeper保存消费者的消费记录的位置信息到zk中。目的是消费者挂了先从zk中读取、在恢复没有成功消费的消息。

0.9之后是保存在本地的。
