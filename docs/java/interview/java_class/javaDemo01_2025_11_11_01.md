# java的class类



![image-20251215172048207](javaDemo01_2025_11_11_01.assets/image-20251215172048207.png)



| 类别           | 方法                                                         | 返回类型         | 用途               | 使用频率 | 简单案例                                             |
| :------------- | :----------------------------------------------------------- | :--------------- | ------------------ | :------- | ---------------------------------------------------- |
| **获取类信息** | `getName()`                                                  | `String`         | 获取完整类名       | ⭐⭐⭐⭐⭐    | `"java.lang.String"`                                 |
|                | `getSimpleName()`                                            | `String`         | 获取简单类名       | ⭐⭐⭐⭐     | `"String"`                                           |
|                | `getPackage()`                                               | `Package`        | 获取包信息         | ⭐⭐⭐      | `java.lang包信息`                                    |
| **实例化相关** | `newInstance()`                                              | `T`              | 创建实例(已过时)   | ⭐⭐       | `new String()`                                       |
|                | `getDeclaredConstructor()`                                   | `Constructor<T>` | 获取构造器         | ⭐⭐⭐⭐     | 构造器对象                                           |
| **字段操作**   | `getField(String name)`                                      | `Field`          | 获取public字段     | ⭐⭐⭐      | 获取publicField                                      |
|                | `getDeclaredField(String name)`                              | `Field`          | 获取所有字段       | ⭐⭐⭐⭐     | 获取privateField                                     |
|                | `getFields()`                                                | `Field[]`        | 获取所有public字段 | ⭐⭐⭐      | `[publicField1, ...]`                                |
|                | `getDeclaredFields()`                                        | `Field[]`        | 获取所有字段数组   | ⭐⭐⭐⭐     | `[field1, field2, ...]`                              |
| **方法操作**   | `getMethod(String name, Class<?>... paramTypes)`             | `Method`         | 获取public方法     | ⭐⭐⭐⭐     | 获取"toString"方法                                   |
|                | `getDeclaredMethod(String name, Class<?>... paramTypes)`     | `Method`         | 获取所有方法       | ⭐⭐⭐⭐     | 获取私有方法                                         |
|                | `getMethods()`                                               | `Method[]`       | 获取所有public方法 | ⭐⭐⭐      | `[method1, method2, ...]`                            |
|                | `getDeclaredMethods()`                                       | `Method[]`       | 获取所有方法数组   | ⭐⭐⭐⭐     | `[所有方法]`                                         |
| **注解相关**   | `getAnnotation(Class<A> annotationClass)`                    | `A`              | 获取注解           | ⭐⭐⭐⭐     | `@Deprecated注解对象`                                |
|                | `getAnnotations()`                                           | `Annotation[]`   | 获取所有注解       | ⭐⭐⭐      | `[@注解1, @注解2]`                                   |
|                | `isAnnotationPresent(Class<? extends Annotation> annotationClass)` | `boolean`        | 判断是否有注解     | ⭐⭐⭐⭐     | `true/false`                                         |
| **类型检查**   | `isInstance(Object obj)`                                     | `boolean`        | 判断是否为实例     | ⭐⭐⭐      | `String.class.isInstance("test") → true`             |
|                | `isAssignableFrom(Class<?> cls)`                             | `boolean`        | 判断是否可赋值     | ⭐⭐⭐      | `Object.class.isAssignableFrom(String.class) → true` |
|                | `isInterface()`                                              | `boolean`        | 判断是否为接口     | ⭐⭐⭐      | `Runnable.class.isInterface() → true`                |

```java
// 示例类定义
class ExampleClass {
    @Deprecated
    public String publicField = "public";
    private String privateField = "private";
    
    public void publicMethod() {}
    private void privateMethod() {}
}

// 测试代码主要输出结果：
// getName() → "ExampleClass"
// getSimpleName() → "ExampleClass" 
// getPackage() → Package对象
// getField("publicField") → Field对象(publicField)
// getDeclaredField("privateField") → Field对象(privateField)
// getFields() → [Field对象(publicField)]
// getDeclaredFields() → [Field对象(publicField), Field对象(privateField)]
// getMethod("publicMethod") → Method对象(publicMethod)
// getDeclaredMethod("privateMethod") → Method对象(privateMethod)
// getMethods() → [toString, hashCode, publicMethod, ...] (包含继承的方法)
// getDeclaredMethods() → [publicMethod, privateMethod]
// getAnnotation(Deprecated.class) → @Deprecated注解对象
// getAnnotations() → [@Deprecated]
// isAnnotationPresent(Deprecated.class) → true
// isInstance(new ExampleClass()) → true
// Object.class.isAssignableFrom(ExampleClass.class) → true
// ExampleClass.class.isInterface() → false
```
