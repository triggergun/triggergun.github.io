## 正确理解` <? extends T> 与 <? super T>`

对于 `<? extends T>` 与 `<? super T>` 一般解释为：

- `<? extends T> `表示对象要么是 T 类型，要么是 T 的子类。
- `<? super T>` 表示对象要么是 T 类型，要么是 T 的父类。
