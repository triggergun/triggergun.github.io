# 学习的几个境界

1. 上课听、刷教学视频、看书懂了叫<font color="red">了解</font>。
2. 刷题会做叫<font color="red">理解</font>。
3. 能给别人道明白其中原理叫<font color="red">掌握</font>。
4. 遇到变式题会解叫<font color="red">运用</font>。
5. 能总结出解题规律叫<font color="red">归纳</font>。
6. 能举一反三触类旁通叫<font color="red">迁移</font>。
7. 发现知识漏洞主动补叫<font color="red">精进</font>。
8. 用所学知识解决实际问题叫<font color="red">实践</font>。
9. 能质疑并优化原有方法叫<font color="red">突破</font>。
