# docker拉取不到镜像问题

## 问题1)使用的是国外镜像源

创建目录：/etc/docker

```shell
sudo mkdir -p /etc/docker
```

在daemon.json文件中写入内容：

```shell
sudo tee /etc/docker/daemon.json <<-'EOF'
{
  "registry-mirrors": ["https://fskvstob.mirror.aliyuncs.com"]
}
EOF
```

守护进程重新加载。

```shell
sudo systemctl daemon-reload
```

重新启动docker。

```shell
sudo systemctl restart docker
```
