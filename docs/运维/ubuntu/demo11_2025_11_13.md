# ubuntu刷新网络配置文件

- 情况1：当我们给ubuntu配置静态IP地址后，需要重新刷新配置文件。我们应该使用`systemctl restart network`命令还是？

在 Ubuntu 中刷新网络配置的命令取决于你使用的网络管理工具和 Ubuntu 版本。以下是几种常见的方法：

## 使用NetworkManager(桌面版常用）

这个ubuntu系统带有系统操作桌面的系统。

```shell
# 重启 NetworkManager 服务
sudo systemctl restart NetworkManager
```

命令会让连接的wifi重新连接呢？

```shell
# 查看当前连接
nmcli connection show --active

# 断开特定连接（将 "WiFi名称" 替换为你的网络名）
sudo nmcli connection down "WiFi名称"

# 重新连接
sudo nmcli connection up "WiFi名称"
```
